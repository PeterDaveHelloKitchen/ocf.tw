開放文化基金會協助台灣開放源碼與開放資料社群舉辦各種活動， 如主題演講、國際交流、大型研討會等，藉由贊助這些活動，除了為台灣開放文化貢獻心力您更有機會能夠……

  • 發掘可能的合作夥伴
  • 招募開源人才
  • 提升企業組織形象及知名度

若您有興趣贊助下列特定類型活動，歡迎與我們聯繫，一起來推動開放文化!


基金會行政基金

本基金會致力於推廣各類開放文化活動，包括: 開放源碼、開放標準、開放硬體、開放資料、開放系統等相關領域之活動。目前針對開放源碼相關社群及活動，提供專戶與收取款行政事務服務，希望將來朝向提供各社群贊助各類特定開發專案或議題研究計劃的財務管道。


國際交流計畫

資助台灣學生出國發表，直接和國際開源社群分享創作成果、經驗交流。受資助者將在回國後舉辦分享會，讓有興趣的社會大眾藉此獲得新知。捐款至本計畫劃將撥款 15% 至基金會行政基金，本計劃所舉辦活動的結餘款， 將撥款 15% 至基金會行政基金， 其餘則投入國際交流計畫之中。

零時政府 g0v 黑客松

g0v 黑客松經費一直由各界自主貢獻，過往由現場小額捐款、香油票、及 g0v 講者捐助演講費支 持。2015 起，大松捐款流程與多個開源社群合力成立的「財團法人開放文化基金會」合作。開放文化基金會將開立捐款收據給捐款人做為抵稅證明，並公開部分捐款人資料，於網路上辦理捐款徵信。捐款人可以選擇公開、部分公開、匿名等方式顯示捐款人身分。捐款總額的 7% 將作為開放文化基金會的行政作業費用。


詳細說明: http://ocf.tw/donate/g0v.html

開放文化影展

開放文化影展籌辦人也是MozTW社群的幕後重要推手趙柏強，在第一屆OCF烤山豬募款餐會中宣布，今年將舉辦開放文化影展，藉由播放開放文化相關的影片，吸引工程師以外的人認識開放文化，而且藉由舉辦電影討論會，更可以促進社群與民眾之間的討論與深化議題等。捐款至本計畫劃將撥款 15% 至基金會行政基金。

協辦大型研討會

台灣一年四季都有各式各樣的研討會，本基金會有能力協助行政業務、轉介相關組織。贊助此項目將有助個人或組織傳播理念、招募新血。


  • 開源人年會 COSCUP
    COSCUP 希望藉由每一屆的年度會議，讓世界各地的 FLOSS 愛好者齊聚一堂，並邀請世界各地的專家上台分享經驗，想法與新的技術，來共同的激發國內外的愛好者更加踴躍投入 FLOSS 的開發與使用。
  • 學生計算機年會 SITCON
    由對資訊抱持熱誠的學生們所組成的同名社群自發籌辦，希望能透過技術與知識的激盪，給予學生們一個用自身力量實踐夢想的舞台。
  • 行動科技應用開發者年會 MOPCON
    MOPCON 專注於行動軟體開發 / 應用技術之專業研討會，也是南部最大社群研討會，希望藉由軟體技術社群的力量，推動南北業界技術實務經驗交流，關注行動應用趨勢，並以軟體科技從業人員出發，定調本年會精神意涵為 “I code, I spread, I connect”。
  • PyCon APAC
    PyCon APAC 是由 Python 社群自發性為了亞太地區 Python 愛好者所舉辦的研討會。PyCon APAC 現在每年舉辦一次，歡迎每一位 Python 使用者參加。PyCon 最初是在 2003 年於北美開始舉辦，而如今全世界都有遵照 PyCon 精神所舉行的會議。
  • JCConf
    Java Community Conference Taiwan 是由社群主辦的 Java 開發者年會，以提供更多機會讓開發人員之間能夠互相交流。
