透過你的捐助支持，可以使我們維持基本的行政運作，並能長期推廣台灣的開放自由文化：

    • 目前刷卡服務可接受VISA、Master Card、JCB信用卡

    • 由基金會負擔手續費每筆2.45%

    • 本頁面已使用 256-bit SSL 安全加密機制

 

[[ 單次捐款 ]]


  • 線上信用卡刷卡

    • 目前刷卡服務可接受VISA、Master Card、JCB信用卡

    • 由基金會負擔手續費每筆2.45%

    • 本頁面已使用 256-bit SSL 安全加密機制

  • Web ATM

    • 由基金會負擔手續費每筆30元、捐款人跨行手續費每筆15元[註]

    • [註] 若您擁有下列任一銀行之金融卡，即可無須負擔跨行手續費。銀行名單：台新銀行、玉山銀行、華南銀行、台灣銀行、台北富邦銀行、第一商業銀行、中國信託、兆豐國際商銀、元大銀行、台灣土地銀行、國泰世華銀行

  • ATM 轉帳

    • 由基金會負擔每筆手續費30元、捐款人跨行手續費每筆15元[註]

    • [註] 若您擁有下列任一銀行之金融卡，可直接選擇該銀行付款，則將可免除跨行手續費。銀行名單：台灣銀行、中國信託、玉山銀行、第一銀行、台北富邦銀行、華南銀行、台新銀行

  • 超商代碼

    • 由基金會負擔每筆30元手續費

    • 需自行記下超商代碼，至超商(7-11/ OK / 全家 / 萊爾富)機台輸入代碼，產生繳費費後至櫃檯繳費

  • 超商條碼繳費

    • 由基金會負擔每筆25元手續費

    • 需自行列印繳費單前往超商(7-11/ OK / 全家 / 萊爾富) 櫃台繳費
